# Tela de Login Bootstrap

Este é um projeto simples de uma tela de login utilizando HTML, Bootstrap e um servidor Node.js com Express.

## Como executar o projeto

1. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
2. No terminal, navegue até a pasta do projeto:
   ```bash
   cd c:\Users\e144231-adm\AnaliseSistemas\HTML\Bootstrap\TelaDeLogin
   ```
3. Instale as dependências:
   ```bash
   npm install express
   ```
4. Inicie o servidor:
   ```bash
   node server.js
   ```
5. Acesse no seu navegador: `http://localhost:3000`
